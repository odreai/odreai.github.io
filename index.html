삶의 안심을 위한 연결고리 · 안심365

데이터 미입력 기반 응급상황 감지 시스템

ODRE AI는 사용자의 ‘데이터 미입력’을 핵심 신호로 감지하여
위험 징후를 예측하고, 보호자에게 자동으로 알림을 전송하는
세계 최초의 다단계 알림 생명 안심 플랫폼입니다.

안심365 다운로드

기술 핵심

• ODRE402 리스크 엔진 — 실시간 위험도 평가 및 주기적 자동 로그 생성
• AI 예측 시스템 — 사용자 패턴 기반 위험 궤적(3일·1주·1개월) 분석
• FCM 양방향 감시 — 사용자 ↔ 보호자 세션·하트비트 실시간 동기화
• 정밀 위치 로그 — 위도·경도 소수점 7자리(약 1.1cm) 단위 저장
• Non-Input 기반 감지 — 움직임·데이터 미입력 자체를 위험 신호로 분석
• 삶의 안심 연결고리 — 혼자 있는 사람을 보호자와 즉시 연결하는 생명 플랫폼

ODRE 보안 핵심 · CryptoEngine 양자내성(PQC) 자동보안 엔진

• CryptoEngine 자동 보안전환: classic → hybrid → PQC 자동 선택
• Kyber / Dilithium 기반 양자내성(PQC) 암호 지원
• AES / RSA / ECDH 기반 클래식 암호화 자동 호환
• 요청 자동 복호화: 무결성 · 서명 · 타임스탬프 전면 검증
• 응답 자동 서명(Signature): 위·변조 완전 차단
• “서버 설정”이 아니라 키 자체가 보안등급을 결정
• 개발자 옵션 없음 — 엔진이 스스로 판단하는 풀 오토 보안
• 사용자의 디바이스 능력에 따라 보안 모드 자동 적용
• 세계적으로도 사례가 드문 완전 자동 Hybrid-PQC 보안 시스템

ODRE CryptoEngine 보안 구조도

Classic Layer
• AES / RSA / ECDH 전통 암호 기반 보호

PQC Layer
• Kyber KEM / Dilithium Signature 기반 양자내성 보호

Hybrid Layer
• Classic + PQC 키 혼합
• 자동 Perfect Forward Secrecy
• Drift Protection(클럭·타임스탬프 변조 감지)

AEAD Layer
• AES-GCM / ChaCha20-Poly1305 데이터 암·복호화

Signature Layer
• Ed25519 / Dilithium 서명 기반 응답 위변조 방지

Canonical Metadata
• nonce, timestamp, client-mode, integrity hash 자동 포함

Drift Protection
• 시간 변조·리플레이 공격 자동 차단

Full Auto Mode
• classic → hybrid → pqc 보안모드 자동 선택
